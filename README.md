# Слевая печать

## Онлайн версия

[Git Book](https://malware-analysis-department.gitbook.io/the-typing/)

## О чем это

Это исчерпывающий гайд по освоению навыка быстрого слепого набора текста.

Для освоения этого материала, нужно иметь в наличии клавиатуру и компьютер с доступом в интеренет. Об остальном позаботится это руководство.

### Предисловие

#### Определеяем текущий уровень

Перед началом пути настоятельно рекомендуется выполнить 2 теста печати и зафиксировать свои результаты: https://monkeytype.com

1. **Обычная печать**

Напечатай небольшой текст так, как обычно делаешь это.

2. **Слепая печать**

Напечататй небольшой текст без подглядывания, постарайтесь как можно точнее нажимать нужные клавиши. Да, скорее всего вы не знаете слепую печать, поэтому вы здесь, тем не менее, важно зафиксировать текущий уровень - медленной напечатайте небольшой текст.&#x20;
